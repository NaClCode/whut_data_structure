## 问题

- 图与景点信息系统
- 连连看游戏

## 运行
### 哈夫曼树
1. [位置](./Huffman)
2. 压缩命令
```shell
./bin/main.exe 文件名 zip
```
3. 解压缩命令(虽然压缩的文件后缀有.huf, 但是解压输入的文件后缀不要带.huf)
```shell
./bin/main.exe 文件名 unzip
```
